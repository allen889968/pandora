通过css border可以实现如上图形，通过不同的宽度等设置，可以实现需要的三角箭头形状(结合最新的CSS3效果，圆角、渐变色以及旋转，可以实现更为强大的需求)，下面是兼容性浅研究

以前实现的时候针对 color为 transparent 的边使用了 border-style:dashed，导致部分电脑的火狐下出现问题，原因是那部分电脑的火狐在解析border时，结果如右图，填充设置颜色透明时，实现css箭头时导致相邻边填充为直线了(而非斜线)，最明显的表现如右二（此问题和火狐在不同电脑上的解析模式可能有所不同导致）

解决办法：使用 solid + transparent 组合即可。

注意：之前实现使用 dashed + transparent 方案是因为IE6，所以解决办法是使用solid但是IE6使用dashed(应用transparent的边)
