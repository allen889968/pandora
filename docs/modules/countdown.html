倒计时插件

实现可定制的倒计时

依赖：使用前请确保页面已经引用了如下文件：

  • http://pic.lvmama.com/min/index.php?f=/js/v5/modules/pandora-countdown.js

老旧版本依赖：

依赖：http://pic.lvmama.com/min/index.php?f=/js/new_v/jquery-1.7.2.min.js,/js/ui/lvmamaUI/lvmamaUI.js

默认传入倒计时间的 UNIX 时间戳 即可

默认效果：
2531275871
253222871
调整时间差：
2531275871
自定义效果：
2531275871
自定义时间格式：
3608000
时间格式自适应：
3608000
自定义结束提醒：
5000

传入参数不正规怎么办？

传入负值
-100000 传入负值，转变为过期提醒
传入文字
传入了文字会怎么样？直接过滤 return了
<span class="time">5000</span>  标签内为倒计时间

<script>
$(function(){
    // 默认调用倒计时 
    $("span.time").countdown();
    
    // 自定义参数说明
    $("span.countdown").countdown({
        format : "dd:hh:mm:ss",   // 时间格式 自定义倒计时类型 现支持 dd:hh:mm:ss(默认) hh:mm:ss dd:hh:mm mm:ss 四种格式
        prezero : true,  // 前导零
        effect : false,  // 支持自定义格式
        overtips : "已结束",  // 自定义结束提醒
        timeauto : false,   // 默认不自适应格式
        timediff : 0   // 调整时间差，单位毫秒
    });
    
    //
    // 老版本调用方法
    //
    $("span.time").ui('countdown');
    
    
    $("span.countdown").ui('countdown',{
        format : "dd:hh:mm:ss",   // 时间格式 自定义倒计时类型 现支持 dd:hh:mm:ss(默认) hh:mm:ss dd:hh:mm mm:ss 四种格式
        prezero : true,  // 前导零
        effect : false,  // 支持自定义格式
        overtips : "已结束",  // 自定义结束提醒
        timeauto : false,   // 默认不自适应格式
        timediff : 0   // 调整时间差，单位毫秒
    });
    
});
</script>
